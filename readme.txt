Weather App
Built with Django 2.2 and python 3.7, sqlite 3, Html and css (Less)
Making API calls from OpenWeatherMap

Steps:
1) Clone the Repo into your local machine (Computer)
2) Create a virtual environment (virtualenv venv)
3) Activate the venv from step 2 (source venv/bin/activate)
4) install the requirements with pip install -r requirements.txt
5) run the App
6) You first have to sign up and sign
7) Test the app by completing the fields on the form
